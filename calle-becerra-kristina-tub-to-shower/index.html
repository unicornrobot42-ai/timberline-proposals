Project Overview

This project involves a full tub-to-shower conversion in the main bathroom at Calle Becerra, Laguna. Timberline Build Co will remove the existing tub, shower, tile wainscot, and flooring, then construct a custom walk-in shower (34" × 60" × 96") complete with euro slider glass enclosure, hot mop shower pan, full tile labor, and all associated plumbing and finish work. The result will be a clean, modern shower built to last.

2
Scope of Work
  • ✦ Demo existing tub, shower, tile wainscot, and bathroom flooring
  • ✦ Remove and reset toilet
  • ✦ Hot mop shower pan (waterproofing)
  • ✦ Construct walk-in shower — 34" × 60" × 96"
  • ✦ Euro slider glass enclosure (standard frameless)
  • ✦ All plumbing — tub-to-shower conversion, toilet reset
  • ✦ Full tile labor — shower walls, floor, and niche
  • ✦ Built-in shower niche
  • ✦ Drywall skim coat and finish paint
  • ✦ Vent fan replacement (material + labor)
  • ✦ Door casing install (labor only — customer supplies material)
  • ✦ Baseboard install (labor only — customer supplies material)
