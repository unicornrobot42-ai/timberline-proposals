Project Overview

Complete Kitchen Transformation

We're doing a full gut-and-rebuild of the kitchen at 650 Avenida Sevilla — demolishing all existing cabinets, sink, and lighting, and installing new cabinetry, quartz countertops, LVP flooring, tile backsplash, and a fully redesigned lighting system across three dedicated dimmer zones.

The layout stays identical to the approved drawings — this is a precision installation with no design changes. All appliances, cabinetry, fixtures, flooring, and tile are owner-supplied unless otherwise noted.

Timberline handles all labor, coordination, HOA package preparation, and permit management.
